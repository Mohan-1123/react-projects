# FoodieHub

A food delivery web app built with React that shows live restaurants from Swiggy.

---

## Tech Stack

![React](https://img.shields.io/badge/React-19-blue?logo=react)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-4-blue?logo=tailwindcss)
![React Router](https://img.shields.io/badge/React_Router-6-red?logo=reactrouter)
![Parcel](https://img.shields.io/badge/Parcel-2-orange)

---

## Features

- Live restaurant listing from Swiggy API
- Search restaurants by name
- Offline detection
- Developer profile from GitHub API
- Client-side routing (Home, About, Contact)

---

## How to Run

```bash
npm install
npm start
```
